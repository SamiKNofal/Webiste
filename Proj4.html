Z BUS MATRIX

Z bus matrix using the buildup algorithm.
Z bus is bus impedance matrix which is used in Fault analysis. This matrix helps in finding the values of fault current. The elements of matrix are the impedances of transmission line connected between two buses. Z Matrix or bus impedance matrix is an important tool in power system analysis. Though, it is not frequently used in power flow study, unlike Y bus matrix, it is, however, an important tool in other power system studies like short circuit analysis or fault study. The Z bus matrix can be computed by matrix inversion of the Y bus matrix. Since the Y bus matrix is usually sparse, the explicit Z bus matrix would by dense so memory intensive to handle directly. By using MATLAB's matrix manipulations, we can create a script with multiple sequential lines of MATLAB commands to develope a program to caluclate the Z bus matrix using the buildup algorithm. The script is written to ensure both efficiency and resiliency providing that the user has no background knowledge of programming or coding.
Code Structure
Before writting the code, we should first understand the procedure behind the buildup algorithm. Provided that a single-line power diagram is available, the user is required to enter both the number of branches and busbars as shown on their system. To specify the connection type between each branch, the following input numbers are used:
  • 1  The branch is connected from a new bus to reference.
  • 2  The branch is connected from an existing bus to new bus.
  • 3  The branch is connected from an existing bus to reference.
  • 4  The branch is connected from a existing bus to another existing bus.
The following MATLAB code summurizes the above discussion:

Once the user specifies the number of branches and busbars, the next step is to iterarte through all branches and request the impedance of each branch as well as the connection type. Both the impedance and connection type are stored on a separate matrices to be used in the calculation step. With each iteration the system shows the user their input impedance and which branch(s) the impedance is located. This provides an insight for the user to insure that their data is stored correctly. A special case is handled when there exist only one branch in the system where the the Z bus matrix is a single 1x1 matrix. The following MATLAB code summurizes this process:

The final step is to calculate the Z bus impedance based on the values and conditions given by the user. Using the buildup algorithm, the system begins to iterarte through each branch and calculate the matrix. The Z bus matrix is calculated irrespective of the system size. Once calculation is done, both the Z bus and Y bus (inverse of Z bus) matrices are given. The following MATLAB code summurizes the final process: 
for i = 2:Branches
    switch Condition(i)
        case 1
            M_Size = size(Zbus);
            Zbus(M_Size(1)+1, M_Size(2)+1) = 0;
            M_Size = size(Zbus);
            Zbus(M_Size(1), M_Size(2)) = Impedance(i);
        case 2
            M_Size = size(Zbus);
            Zbus(:, M_Size(2)+1) = Zbus(:, M_Size(2));
            Zbus(M_Size(1)+1, :) = Zbus(M_Size(1), :);
            M_Size = size(Zbus);
            Zbus(M_Size(1), M_Size(2)) = 0;
            Zbus(M_Size(1), M_Size(2)) = Impedance(i)+Zbus(M_Size(1)-1, M_Size(2)-1);
        case 3
            M_Size = size(Zbus);
            Zbus(:, M_Size(2)+1) = Zbus(:, M_Size(2));
            Zbus(M_Size(1)+1, :) = Zbus(M_Size(1), :);
            M_Size = size(Zbus);
            Zbus(M_Size(1), M_Size(2)) = 0;
            Zbus(M_Size(1), M_Size(2)) = Impedance(i)+Zbus(M_Size(1)-1, M_Size(2)-1);
            M_Size = size(Zbus);
            for k = 1:(M_Size(1)-1)
                for j = 1:(M_Size(1)-1)
                    Zbus(k, j) =  Zbus(k, j) - ((Zbus(k, M_Size(1)))*(Zbus(M_Size(1), j))/Zbus(M_Size(1), M_Size(2)));
                end
            end
            Zbus(M_Size(1), :) = [];
            Zbus(:, M_Size(2)) = [];
        case 4
            j = BusBarCon(Alpha+1);
            Alpha = Alpha+1;
            k = BusBarCon(Alpha+1);
            Alpha = Alpha+1;
            M_Size = size(Zbus);
            Zbus(:, M_Size(2)+1) = Zbus(:, j) - Zbus(:, k);
            Zbus(M_Size(1)+1, :) = Zbus(j ,:) - Zbus(k ,:);
            M_Size = size(Zbus);
            Zbus(M_Size(1), M_Size(2)) = Zbus(j, j) + Zbus(k, k) + Impedance(i) - 2*Zbus(j, k);
            M_Size = size(Zbus);
            for k = 1:(M_Size(1))
                for j = 1:(M_Size(1))
                    Zbus(k, j) =  Zbus(k, j) - ((Zbus(k, M_Size(1)))*(Zbus(M_Size(1), j))/Zbus(M_Size(1), M_Size(2)));
                end
            end
            Zbus(M_Size(1), :) = [];
            Zbus(:, M_Size(2)) = [];
    end 
end
