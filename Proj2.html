CERBERUS APP

Cerberus Drone App.
Enhancing the controllability and functionality of Cerberus Drone is an important aspect to offer users ease of accessability and customization. One way to achieve this goal is to provide a dedicated mobile companion app for the system to fully access the features of the drone on the go. This can be done by developing applications on the Android operating system. Android is a mobile operating system developed by Google, based on a modified version of the Linux kernel and other open source software and designed primarily for touchscreen mobile devices such as smartphones and tablets. Applications are written using the Android software development kit (SDK) and, often, the Java programming language is used. To develop Android applications the Android Studio software tool is used. Android Studio is the integrated development environment (IDE) for Google's Android operating system, built on JetBrains' IntelliJ IDEA software and designed specifically for Android development.
By developing an Android App, namely Cerberus Drone App, it allows users to control multiple on-board electronic systems within the quadcopter, display a graphical representation of sensors' data, use the GPS functionality to pinpoint the device's location, configure the system's parameters and provide a backup virtual analog controller feature to allow resiliency in case any problem occur with the main hand-held controller as well as using it for prototype and testing purposes. Additional features can be added such as storage of data recieved from the drone as XML file formate, displaying weather forcast to notify the user if the weather conditions are suitable for flying the drone or not and other enhanced features that may prove to be useful for the user's experience.
This added application feature also display the programming skills gained from job experince and how I applied them for Cerberus Drone project to combine both software and hardware knowledge.
